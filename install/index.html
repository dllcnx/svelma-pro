如何使用



1

NPM

此组件库适用于基于rollup或者webpack搭建的svelte及sapper项目

下载安装

下载bluma与组件库.
集成scss模板语法支持(因为组件库采用了类似vue的scss模板写法,所以需要也集成进去).
添加 scss编译 到你的 rollup.config.js 配置文件中(或者webpack.config.js).
如果使用webpack,在 webpack.config.js 配置文件中.
